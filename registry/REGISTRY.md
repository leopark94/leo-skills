## 설치 가이드

### 개별 스킬 설치

```bash
# 1. 클론
gh repo clone <owner>/<repo> /tmp/<repo>

# 2. 스킬 파일 복사
cp -r /tmp/<repo>/skills/<skill-name> ~/.claude/skills/

# 3. 또는 심링크
ln -sf /tmp/<repo>/skills/<skill-name> ~/.claude/skills/<skill-name>
```

### 플러그인으로 설치 (지원하는 레포)

```bash
# Claude Code 플러그인 시스템
claude plugin add <github-url>
```

### leo-skills에서 관리

```bash
# 검색
leo skills search security

# 설치
leo skills install trailofbits/skills

# 목록
leo skills list
```
